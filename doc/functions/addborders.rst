AddBorders
==========

.. function::   AddBorders(clip clip[, int left=0, int right=0, int top=0, int bottom=0, float[] color=[0, 0, 0]])
   :module: std
   
   Adds borders to frames. The arguments specify the number of pixels to add on each side. They must obey the subsampling restrictions.
   The newly added borders will be set to *color*.
